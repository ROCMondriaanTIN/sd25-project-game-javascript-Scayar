üìê Architectuur Overzicht

MVC Pattern:
‚Ä¢ Model (hogerlager.js): Bevat game state en logica. Geen DOM interactie.
‚Ä¢ View (index.html + CSS): Definieert HTML structuur en styling. Geen logica.
‚Ä¢ Controller (controller.js): Handelt events af, communiceert met model, update view.

Data Flow

1. User klikt button ‚Üí Controller event handler 2. Controller roept Model functie aan 3. Model update state en retourneert data 4. Controller ontvangt data 5. Controller update View (DOM) 6. User ziet resultaat

Game Flow

1. Computer gooien ‚Üí startNewRoundWithComputerRoll() 2. Speler kiest inzet + voorspelling ‚Üí setBetAndPrediction() 3. Speler gooien ‚Üí rollPlayerAndResolve() 4. Resultaat wordt getoond ‚Üí renderResultMessage() 5. Geschiedenis wordt bijgewerkt ‚Üí renderHistory()
